Knockout Instant Demo

Simple example

Use the instant binding on any input element, that gets values from typing (such as input[type=text] or textarea).

ko.applyBindings({
  demoValue: ko.observable()
});
<input type='text' data-bind='instant: demoValue'>
<p>
  <strong>Value:</strong>
  <span data-bind='text: demoValue'></span>
</p>
